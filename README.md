# Aplikasi Akuntansi Sederhana (Web, LocalStorage)

Aplikasi akuntansi sederhana berbasis web yang menyimpan data di LocalStorage browser, mengikuti `coding-plan.md`.

## Cara Menjalankan

- Buka file `index.html` langsung di browser modern (Chrome, Firefox, Edge, Safari).
- Tidak diperlukan server atau database.

## Fitur Utama (versi awal)

- Input transaksi harian (debit/kredit sederhana).
- Jurnal umum otomatis dari transaksi.
- Buku besar per akun.
- Trial balance (validasi total debit/kredit).
- Laporan laba rugi dan neraca sederhana.

Backup dan restore data tersedia melalui tombol **Export Data** dan **Import Data** di navbar.
